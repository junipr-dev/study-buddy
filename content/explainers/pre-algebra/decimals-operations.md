# Operations with Decimals

## What are Decimals?

Decimals are another way to write fractions, using place value and a decimal point. Each place to the right of the decimal point represents a power of 10.

**Place values:**
```
3.1415
│││││
│││││__ Thousandths (1/10,000)
││││___ Hundredths (1/100)
│││____ Tenths (1/10)
││_____ Ones
│______ Decimal point
```

Examples:
- 0.5 = 5/10 = 1/2
- 0.75 = 75/100 = 3/4
- 3.14 = 3 and 14/100

## Adding and Subtracting Decimals

**Rule:** Line up the decimal points, then add or subtract as normal.

### How to Add/Subtract:

1. **Write numbers vertically** - One above the other
2. **Line up decimal points** - Make sure they're aligned
3. **Add zeros if needed** - To make same number of decimal places
4. **Add or subtract** - Work right to left as usual
5. **Bring down decimal** - Decimal point stays in same position

## Step-by-Step Example (Addition)

**Add:** 3.45 + 12.8

### Step 1: Line up decimal points
```
   3.45
+ 12.8
```

### Step 2: Add zero to make even
```
   3.45
+ 12.80
```

### Step 3: Add from right to left
```
   3.45
+ 12.80
-------
  16.25
```

### Step 4: Check your answer
```
3.45 + 12.8 = 16.25
Approximately: 3 + 13 = 16 ✓
```

## Step-by-Step Example (Subtraction)

**Subtract:** 5.6 - 2.38

### Step 1: Line up decimals
```
  5.6
- 2.38
```

### Step 2: Add zero
```
  5.60
- 2.38
```

### Step 3: Subtract (borrow if needed)
```
  5.60
- 2.38
------
  3.22
```

**Check:** 2.38 + 3.22 = 5.60 ✓

## Multiplying Decimals

**Rule:** Multiply as if there are no decimals, then count total decimal places.

### How to Multiply:

1. **Ignore decimals** - Multiply as whole numbers
2. **Count decimal places** - In both numbers being multiplied
3. **Place decimal** - That many places from the right in answer
4. **Add zeros if needed** - If not enough digits

## Step-by-Step Example (Multiplication)

**Multiply:** 2.5 × 1.3

### Step 1: Multiply without decimals
```
  25
× 13
----
  75  (25 × 3)
250  (25 × 10)
----
325
```

### Step 2: Count decimal places
```
2.5 has 1 decimal place
1.3 has 1 decimal place
Total: 1 + 1 = 2 decimal places
```

### Step 3: Place decimal
```
325 → 3.25 (2 places from right)
```

**Check:** 2.5 × 1.3 ≈ 2.5 × 1 = 2.5, close to 3.25 ✓

## Dividing Decimals

**Rule:** Move decimal point in divisor to make it a whole number, then move decimal same places in dividend.

### How to Divide:

1. **Move decimal in divisor** - Make it a whole number
2. **Move decimal in dividend** - Same number of places
3. **Place decimal in answer** - Directly above
4. **Divide normally** - As you would with whole numbers

## Step-by-Step Example (Division)

**Divide:** 6.4 ÷ 0.8

### Step 1: Move decimal in divisor (0.8 → 8)
Move 1 place right

### Step 2: Move decimal in dividend too (6.4 → 64)
Move 1 place right

### Step 3: Divide
```
64 ÷ 8 = 8
```

**Check:** 8 × 0.8 = 6.4 ✓

## Converting Between Fractions and Decimals

**Fraction to decimal:** Divide numerator by denominator
```
3/4 = 3 ÷ 4 = 0.75
1/8 = 1 ÷ 8 = 0.125
```

**Decimal to fraction:** Use place value
```
0.6 = 6/10 = 3/5
0.25 = 25/100 = 1/4
```

## Key Points to Remember

✓ **Line up decimals** - For addition and subtraction
✓ **Count total decimal places** - For multiplication
✓ **Move decimals together** - For division
✓ **Add zeros freely** - Doesn't change value (3.5 = 3.50)
✓ **Multiply by 10, 100, 1000** - Just move decimal right
✓ **Divide by 10, 100, 1000** - Just move decimal left

## Common Mistakes to Avoid

❌ **Not lining up decimals** - Wrong: Add 3.45 + 12.8 by lining up the 8 under the 5
✓ Correct: Line up decimal points

❌ **Forgetting to count decimal places** - Wrong: 2.5 × 1.3 = 325
✓ Correct: 2.5 × 1.3 = 3.25 (2 decimal places)

❌ **Moving only one decimal** - Wrong: 6.4 ÷ 0.8, move decimal in 0.8 only
✓ Correct: Move decimal same amount in both numbers

❌ **Decimal point in wrong place** - Always estimate to check!
✓ Correct: Use estimation to verify answer is reasonable
