# Simplifying Radical Expressions

## What are Radical Expressions?

A radical expression contains a square root, cube root, or other root. The most common is the square root (√).

**Parts of a radical:**
- √a where a is the radicand (number under the root)
- ∛a = cube root
- ⁿ√a = nth root

Examples:
- √16 = 4
- √50 = 5√2
- 2√3 + 3√3 = 5√3

## Simplifying Square Roots

**Goal:** Get perfect squares out from under the radical

**Steps:**
1. **Factor the radicand** - Find perfect square factors
2. **Take square root of perfect squares** - Move outside radical
3. **Leave rest inside** - Non-perfect squares stay
4. **Simplify** - Multiply coefficients outside

## Step-by-Step Example

**Simplify:** √72

### Step 1: Factor 72
```
72 = 36 × 2
   = 6² × 2
```

### Step 2: Separate perfect square
```
√72 = √(36 × 2)
    = √36 × √2
```

### Step 3: Simplify perfect square
```
= 6√2
```

### Check
```
6√2 ≈ 6 × 1.414 ≈ 8.485
√72 ≈ 8.485 ✓
```

## More Examples

### Example 1: With variables
**Simplify:** √(18x³)

```
18 = 9 × 2 = 3² × 2
x³ = x² · x

√(18x³) = √(9 · 2 · x² · x)
        = √9 · √x² · √(2x)
        = 3x√(2x)
```

### Example 2: Large number
**Simplify:** √200

```
200 = 100 × 2 = 10² × 2

√200 = √(100 × 2)
     = 10√2
```

### Example 3: Multiple perfect squares
**Simplify:** √(48x⁴y²)

```
48 = 16 × 3 = 4² × 3
x⁴ = (x²)²
y² = (y)²

√(48x⁴y²) = 4x²y√3
```

## Adding and Subtracting Radicals

**Rule:** Can only combine like radicals (same radicand)

### Example: Addition
**Simplify:** 3√5 + 7√5

```
Like radicals (both √5):
3√5 + 7√5 = 10√5
```

### Example: After simplifying
**Add:** √12 + √27

```
Simplify each:
√12 = √(4 × 3) = 2√3
√27 = √(9 × 3) = 3√3

Now combine:
2√3 + 3√3 = 5√3
```

## Multiplying Radicals

**Rule:** √a · √b = √(ab)

### Example: Multiplication
**Multiply:** √3 · √12

```
Method 1: Multiply first
√3 · √12 = √(3 × 12) = √36 = 6

Method 2: Simplify first
√12 = 2√3
√3 · 2√3 = 2 · √3 · √3 = 2 · 3 = 6
```

### Example: With coefficients
**Multiply:** 2√5 · 3√10

```
Multiply coefficients: 2 × 3 = 6
Multiply radicals: √5 · √10 = √50 = 5√2

Result: 6 · 5√2 = 30√2
```

## Dividing Radicals

**Rule:** √a / √b = √(a/b)

### Example: Division
**Divide:** √50 / √2

```
√50 / √2 = √(50/2) = √25 = 5
```

## Rationalizing the Denominator

**Goal:** No radicals in denominator

**Method:** Multiply by special form of 1

### Example: Simple rationalization
**Rationalize:** 5/√3

```
Multiply by √3/√3:
5/√3 · √3/√3 = 5√3/3
```

### Example: Binomial denominator
**Rationalize:** 6/(2 + √3)

```
Multiply by conjugate: (2 - √3)/(2 - √3)

= 6(2 - √3)/[(2 + √3)(2 - √3)]
= 6(2 - √3)/(4 - 3)
= 6(2 - √3)/1
= 12 - 6√3
```

## Higher Roots

### Cube Roots

**Simplify:** ∛54

```
54 = 27 × 2 = 3³ × 2

∛54 = ∛(27 × 2)
    = 3∛2
```

## Key Points to Remember

✓ **Perfect squares:** 1, 4, 9, 16, 25, 36, 49, 64, 81, 100...
✓ **Factor completely** - Find all perfect square factors
✓ **Like radicals only** - For adding/subtracting
✓ **Multiply/divide freely** - Any radicals
✓ **Rationalize denominators** - No √ in bottom
✓ **Simplify completely** - Get all perfect squares out

## Common Mistakes to Avoid

❌ **Adding unlike radicals** - Wrong: √2 + √3 = √5
✓ Correct: √2 + √3 stays as is

❌ **Distributing square root** - Wrong: √(a + b) = √a + √b
✓ Correct: Cannot split addition/subtraction under radical

❌ **Forgetting to simplify** - Wrong: √12 as final answer
✓ Correct: √12 = 2√3

❌ **Leaving radical in denominator** - Wrong: 5/√2
✓ Correct: 5√2/2
